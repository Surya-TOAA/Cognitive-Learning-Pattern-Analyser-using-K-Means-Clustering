1. SCROLL-TRIGGERED REVEALS:
   Use IntersectionObserver on every section/feature.
   When element enters viewport (threshold: 0.15):
   - Add class .visible
   - CSS: .visible { opacity: 1; transform: translateY(0); }
   - Default state: opacity: 0; transform: translateY(40px); transition: all 0.6s ease;
   - Stagger children using animation-delay: 0.1s, 0.2s, 0.3s...

2. TYPING ANIMATION:
   - Split text into characters with a JS function
   - Each char wrapped in a span: opacity 0, translateY 8px
   - Animate in sequence with 30ms delay per character
   - Show blinking cursor during animation, hide after
   - Trigger on page load for hero; trigger on IntersectionObserver for sections

3. PARTICLE CANVAS:
   - Hero canvas: 200 particles, interactive repel on mouse
   - Section 7 halves: 50 particles each, no interaction
   - CTA section: 120 particles, static color palette
   - Use requestAnimationFrame loop

4. CUSTOM CURSOR on sliders:
   - Show a pill-shaped div ("▶ View Story") on mouseover of slider
   - Track mousemove and update transform: translate(x, y) with CSS transition
   - Scale from 0 on enter, scale to 0 on leave

5. SLIDER DRAG:
   - Use mousedown/mousemove/mouseup + touchstart/touchmove/touchend
   - Detect swipe direction, advance slide by 1
   - Snap to nearest slide on release

6. HEADER BEHAVIOR:
   - Transparent on hero, frosted glass after scrolling 50px
   - Hide header on scroll down (add .hidden class: translateY(-100%))
   - Show header on scroll up (remove .hidden)

7. FEATURE BLOCK VISUAL ANIMATIONS:
   - Forgetting curve: SVG path draws itself with stroke-dasharray + dashoffset
   - Radar chart: SVG polygon points morph between two sets of values (lerp in JS loop)
   - Progress bars: width animates from 0 to value when section enters viewport
   - Session cards: pulse animation on status dot
