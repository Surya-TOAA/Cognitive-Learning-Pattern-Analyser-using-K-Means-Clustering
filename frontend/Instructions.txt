This view uses the Claude API to generate real insights.

Top: "Generate Insights" button — clicking this calls the Claude API with the 
learner's mock data payload and streams the response.

API call structure:
  - Model: claude-sonnet-4-6
  - System prompt: "You are an expert learning coach and cognitive scientist analyzing 
    a student's learning data. Generate specific, actionable insights about their 
    learning patterns, cognitive style, retention gaps, and optimization recommendations. 
    Format your response as JSON with an array of insight objects, each having: 
    type (Pattern|Alert|Trend|Recommendation), title, body, priority (high|medium|low), 
    and actionItems array."
  - User message: Pass the mock learner data as JSON

Display insights as cards:
  - Icon based on type (🧠 Pattern, ⚡ Alert, 📈 Trend, 🎯 Recommendation)
  - Priority badge (high=red, medium=amber, low=green)
  - Title + body text
  - Expandable "Action Items" list
  - Dismiss button, "Implement" button

Also include a free-form AI Chat section at the bottom:
  - Input: "Ask your AI learning coach anything..."
  - Sends to Claude API with learner context in system prompt
  - Renders responses in a chat bubble UI (user right, AI left)
  - Show typing indicator while streaming
